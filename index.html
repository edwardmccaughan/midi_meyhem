Midi meyhem

plug in a midi keyboard and visualise the sounds
press number keys 1-7 to switch visualisations. If you don't have a midi keyboard, the qwerty keys will similute at least some of the piano keys
  • between_worlds
  • pixi_radiant
  • so_many_vs
  • voronoi_sparkles
  • walkers
  • waaaaaaves
  • spiralize
  • dots_everywhere

source code is at https://github.com/edwardmccaughan/three_midi_fireworks_noodling
disclaimer: the Web MIDI API only works in chrome and opera for now, although the regular keyboard will still work.
